# 🍕 MsPizza - Site de Pizzaria

Projeto de site para uma pizzaria chamado **MsPizza**, desenvolvido com **HTML, CSS e JavaScript**, com sistema de **carrinho de compras** e páginas de navegação para cardápio e contato.

---

## 📌 Sobre o Projeto

O **MsPizza** é um site moderno de pizzaria criado para apresentar:

* Página inicial com destaque da pizzaria
* Cardápio de pizzas
* Página de contato
* Sistema de carrinho de compras
* Interface simples e responsiva

O objetivo é simular um **site real de pedidos de pizza online**.

---

## 🚀 Como Executar o Projeto

1. Baixe ou clone o repositório:

```
git clone https://github.com/seu-usuario/mspizza.git
```

2. Abra a pasta do projeto.

3. Execute o arquivo:

```
index.html
```

4. Para usar o **carrinho em PHP**, rode em um servidor local como:

* XAMPP
* WAMP
* Laragon
